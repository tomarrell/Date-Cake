# DateCake
A natural-language date calculator.

## Usage

Useful commands:
- ````add 20 days to March 2nd```` (will default to current year if none specified)
- ````what is 2m 23d 5h subtracted from 18/08/2017````
- ````what day will it be 20d from Jan 5````
- ````what day was that````
